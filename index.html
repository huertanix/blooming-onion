Blooming Onion

Shell script for automating the deployment of a Ghost.org blog as a Tor .onion service.

Download as .zip Download as .tar.gz View on GitHub

About

The internet as it exists is a double-edged sword of speech and surveillance. Its protocols allow your blog posts to travel freely to a global audience, but at the price of surrendering your anonymity to that audience through the mechanisms of ICANN when you register a domain name. Tor onion (aka "hidden") services offer an alternative to that paradigm by allowing anyone to create a web presence on the Tor ~darknet~ without giving up any of your deets to ICANN or your blog's IP address to anyone at all. Self-hosted blogs and onion services take some skills to set up, so we created this script to let you run a DigitalOcean 1-Click Application for Ghost blog as a Tor .onion service to make it easy to set up completely from inside the Tor Browser. To see an example, go to this .onion address in the Tor Browser: v4uvcyaykxrfbxdn.onion.
